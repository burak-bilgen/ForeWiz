# Changelog

## [1.0.0] - Initial Release

### Added
- WeatherKit integration for real-time weather data
- Widget extension for home screen weather insights
- CoreLocation for location-based weather
- StoreKit subscription system
- Google AdMob integration with GDPR consent
- Turkish and English localization
- Dark mode support
- Background refresh with BGTaskScheduler
- SwiftData persistence layer
- Decision engines for outfit/activity recommendations

### Fixed
- Force unwrap crash in WeatherInsightsView
- Timer memory leaks in loading animations
